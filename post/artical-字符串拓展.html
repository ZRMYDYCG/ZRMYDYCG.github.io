字符串拓展

split 方法

它与 join 相反，它是将字符串转换成数组

代码示例 👉

<script>
    const str = 'pink,red'
    const arr = str.split('')
    console.log(arr)
</script>

---控制台---
Array(8) ['p','i','n','k',',','r','e','d']

咱现在传入分隔符试试：

<script>
    const str = 'pink,red'
    const arr = str.split('-')
    const arr1 = str.split(',')

    console.log('-分隔', arr);

    console.log('----------')

    console.log(',分隔', arr1)
</script>

---控制台---
Array(1) ["pink,red"]
----------
Array(2) ["pink","red"]

substring 方法

语法 👉

str.substring(indexStart[,indexEnd])

返回值: 咱截取的字符串

indexStart

需要截取的第一个字符的索引，该索引位置的字符作为字符串的首字母

indexEnd

可选,一个 0 到字符串长度之间的整数,以该数字为索引的字符不包含在截取的字符串内。如果省略不写的话,就是指截取 indexStart 索引到最后一个字符。

代码示例 👉

<script>
    const str = '不积跬步,无以至千里'

    let sub = str.substring(5)

    console.log(sub)  // output: 无以至千里
</script>

startsWith

Vue 路由部分要用到噢~

stratWith()方法用来判断当前字符串是否以另外一个给定的字符串开头,并根据判断结果返回 true or false。

语法：str.startsWith(searchString[,position])

参数

searchString : 要搜索的字符串

position（可选）: 在 str 中搜索 searchString 的开始位置,默认值为 0

代码示例 👉

var str = "To be, or not to be, that is the question.";

alert(str.startsWith("To be"));         // true
alert(str.startsWith("not to be"));     // false
alert(str.startsWith("not to be", 10)); // true

endsWith

和上面相反：从字符串的后面开始搜索

includes

includes() 方法执行区分大小写的搜索，以确定是否可以在另一个字符串中找到一个字符串，并根据情况返回 true 或 false。

语法：str.includes(searchString[,position])

参数

searchString:要在此字符串中搜索的字符串

position:规定要从当前字符串哪个索引位置开始搜寻子字符串，默认值为 0

代码示例 👉

const str = "To be, or not to be, that is the question.";

console.log(str.includes("To be")); // true
console.log(str.includes("question")); // true
console.log(str.includes("nonexistent")); // false
console.log(str.includes("To be", 1)); // false
console.log(str.includes("TO BE")); // false
console.log(str.includes("")); // true
