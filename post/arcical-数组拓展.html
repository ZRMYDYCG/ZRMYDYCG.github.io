数组扩展

数组扩展
|—— `...` 展开运算符十分重要，有着复制，合并数组的作用，也可以和数组解构赋值进行结合
|—— `Array.from` 也是十分重要的，它可以将类数组转化为数组
|—— `Array.of` 这个方法了解一下，其实就是一种创建数组的方式，也是Array创建数组的一个改进
|—— `find` 和 `findIndex` 方法就是查找数组中第一个符合条件的的元素，属于是数组实例身上的方法
|—— `fill` 方法是很重要的，它可以对数组进行填充和替换
|—— `flat() flatMap` 方法是数组的扁平化处理，其中flatMap可以实现多维嵌套数组的处理，将对象中的数组抽取出来实现扁平化。

... 展开运算符

作用：

快速地进行数组的复制，数组的合并，能够与解构进行结合。

数组的复制（注意是 浅复制 ）

咱以前的复制方案 👉

<script>
    // 1-(...)展开运算符
    let arr = [1, 2, 3]
    let arr2 = arr
    let arr3 = []
    for (let i = 0; i < arr.length; i++) {
        arr3[i] = arr[i];
    }
    // concat:连接字符串  concat() 参数为空不连
    let arr4 = arr.concat()
    console.log('1.', arr2, '2.', arr3, '3.', arr4);
</script>

---控制台---
打印出 3 个一样的数组

使用 … 👉

<script>
    let arr = [1, 2, 3]
    let arr2 = [...arr]
    arr2.pop()
    console.log(arr, arr2);
</script>

使用…(展开运算符)复制，且不会改变原来的数组 arr，这样来看这个（[…arr]），有没有一种就是把 arr 这个数组中括号打烂来，放到新家的感觉

数组的合并

concat方案 👉

<script>
    let arr1 = [1, 2, 3]
    let arr2 = [4, 5, 6]
    let arr3 = arr1.concat(arr2)
    console.log(arr3)
</script>

那试试咱学地这个展开运算符 👉

<script>
  let arr1 = [1, 2, 3]
  let arr2 = [4, 5, 6]
  let arr3 = [...arr1, ...arr2]
  console.log(arr3) // [1,2,3,4,5,6]
</script>

与解构进行结合

<script>
    let arr1 = [1, 2, 3, 4, 5, 6, 7, 8]
    let [a, b, c] = arr1
    console.log(a, b, c) // 1 2 3
</script>

那 … 的用武之处在哪里呢？咱这样写试试 ：[a,b,...c] = arr1 , 咱在控制台就发现这个 c 接受了剩余元素形成了一个数组

注意

... 在解构中的使用,只能放在最后一个元素身上,也就是把前边的元素被特定的元素取走,剩余的交个最后一个元素。

Array.from

首先讲讲这个 arguments 方法，arguments 方法能够在没有形参的情况下拿到所有实参的内容，但是 arguments 不能使用数组的方法，它是类数组。

代码示例 👉

<script>
  function test() {
      console.log(arguments);
      console.log(arguments.pop());
  }
  test(1, 2, 3, 4)
</script>

---控制台---
Arguments(4)
Uncaught TypeError: arguments.pop() is not a function

于是乎，这个 Array.from() 就可以将这些类似数组的东西转化成数组

代码示例 👉

<script>
  function test() {
      console.log(Array.from(arguments));
  }
  test(1, 2, 3, 4)
</script>

---控制台---
Array(4)

再比如这个节点数组

代码示例 👉

<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
</ul>

<script>
  let list = document.querySelectorAll('ul li')
  console.log(list);
  list.pop()
</script>

---控制台---
NodeList(4)
Uncaught TypeError: list.pop() is not a function

那咱用Array.from()将其转换一下，再用fliter方法将下标大于 2 的 li 节点打印出来

<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
</ul>

<script>
  let list = document.querySelectorAll('ul li')
  let arr = Array.from(list).filter(function (item, index) {
      return index > 2
  })
  console.log(arr);
</script>

Array.of

Array.of 方法可以弥补 Array 方法创建数组的短板，也就是当 Array() 里边只传一个参数的话，就是生成该参数个数的空数组。

代码示例 👉

<script>
  let arr = [1, 2, 3]
  let arr1 = Array(3)
  let arr2 = Array.of(1, 2, 3)
  console.log('1',arr,'2', arr1,'3', arr2);
</script>

---控制台---
1. Array(3) [1,2,3] 
2. Array(3) [] length 为 3 的空数组
3. Array(3) [1,2,3]

flat() flatMap()

可以将嵌套的数组拉平，变成一维的数组

代码示例 👉

<script>
    let arr = [1, 2, 3, [4, 5, 6]]

    // 进行数组的扁平化处理 将原来的二维的嵌套数组转化为一维数组
    let arr1 = arr.flat()

    console.log('1',arr,'2',arr1);
</script>

---控制台---
1. Array(4) [1, 2, 3, [4, 5, 6]]
2. Array(6) [1,2,3,4,5,6]

那当场景逐渐变得复杂了呢？比如 👉

<script>
    let arr = [
        {
            name: "夜晚",
            list: ["🎑", "🌠", "🌚"]
        },
        {
            name: "白天",
            list: ["⛈️", "🌤️", "🌝"]
        }
    ]

    let arr1 = arr.flat()

    console.log(arr1);
</script>

其实ES6考虑地很周到，可以用 flatMap() 方法来实现这种嵌套的关系

代码示例 👉

<script>
    let arr = [
        {
            name: "夜晚",
            list: ["🎑", "🌠", "🌚"]
        },
        {
            name: "白天",
            list: ["⛈️", "🌤️", "🌝"]
        }
    ]
    let res = arr.flatMap(function (item) {
        return item.list
    })

    console.log(res);
</script>

---控制台---
Array(6) ["🎑", "🌠", "🌚","⛈️", "🌤️", "🌝"]

fill 方法

是一种数组填充替换方法

<script>
  // 填充:
  let arr = new Array(3).fill("hello")

  console.log('1',arr);

  // 替换: 后边继续添加参数 从哪个开始 从哪个结束
  let arr1 = [11, 22, 33, 44, 55, 66]

  console.log('2',arr1.fill('替换值', 0, 3))
</script>

---控制台---
1. Array(3) ["hello","hello","hello"]

2. Array(6) ["替换值","替换值","替换值","","",""]

find findIndex（ findLast findLastindex）

find 方法就是从数组中找到符合条件的第一个数组元素

findIndex 方法便是从数组中找到符合条件的第一个数组元素的下标

findLast findLastIndex 顾名思义就是从该数组的后面开始

代码示例 👉

<script>
    let arr = [11, 12, 13, 14, 15]

    let res = arr.find(function (item) {
        return item > 13
    })

    let resIndex = arr.findIndex(function (item) {
        return item > 13
    })

    console.log(res, resIndex);
</script>
