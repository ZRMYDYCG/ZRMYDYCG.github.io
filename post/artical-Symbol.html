Symbol

ES6 引入了一种新的原始数据类型 Symbol，表示独一无二的值。它是JavaScript的第七种数据类型，是一种类似于字符串的数据类型。其他数据类型是: undefined、null、布尔值(Boolean) 、字符串 (String)、数值 (Number)、对象(Object)。

Symbol 的特点

  1. Symbol 的值是唯一的，用来解决命名冲突的问题
  2. Symbol 值不能与其他数据进行运算
  3. Symbol 定义的对象属性不能使用 for…in 循环遍历，但是可以使用 Reflect.ownKeys 来获取对象的所有键名

通过函数调用，创建 Symbol

Symbol 的创建与介绍

第一种创建方式 👉

let s1 = Symbol();
console.log(s1,typeof s1);

---控制台---
Symbol() "symbol"

既然是函数式创建,那么可不可以传入值呢？当然是可以的

let s2 = Symbol('创建Symbol');

这里面传入的值，实际上是一个起描述性的作用的字符串，相当于是做了一个注释。

比如 👉

let s2 = Symbol('创建Symbol');
let s3 = Symbol('创建Symbol');
console.log(s2 === s3);

---控制台---
false

第二种创建方式 👉

let s4 = Symbol.for('创建Symbol');
console.log(s4,typeof s4);

---控制台---
Symbol('创建Symbol') "symbol"

用该方式方式创建（可以把Symbol.for()看作是函数对象），可以通过这个描述值，得到两个相同的 symbol 值。

let s4 = Symbol.for('创建Symbol');
let s5 = Symbol.for('创建Symbol');
console.log(s4 === s5);

---控制台---
true

Symbol 的使用

Symbol 的一个使用场景就是来给对象添加属性和方法，表示该属性或方法是独一无二的。

代码示例 👉

向对象中添加方法

写法一

比如我们想要向里面添加方法，但不清楚这个对象里面有什么属性和方法。用如下方式是不安全的:

let game = {
    ...
}
game.up = function(){
    console.log('哈哈哈哈')
}

所以我们可以借助 Symbol 来完成这个事情，Symbol 可以更加简单高效且安全。

let game = {
    name: '俄罗斯方块'，
    down:function() {

    } ,
    up:function() {
        
    } ,
}
// 声明一个对象
let methods = {
    up: Symbol(),
    down: Symbol()
}
// 给 game 扩展方法

game[methods.up] = function() {
    console.log('我是up')
}

game[methods.down] = function() {
    console.log('我是down')
}

console.log(game)

---控制台---
down: f()
up: f()
name:"俄罗斯方块"
Symbol():f()
Symbol():f()

这样就安全地向对象中添加了两个方法。

写法二

代码示例

let geme = {
    name:"吃屎",
    [Symbol('nice')]: function() {
        console.log('真香!')
    },
    [Symbol('bag')]: function() {
        console.log('丁真')
    }
}

Symbol 的内置属性

除了定义自己使用的 Symbol 值以外，ES6 还提供了 11 个内置的 Symbol 值，指向语言内部使用的方法

MDN

https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Symbol

举例

内置的Symbol.isConcatSpreadable符号用于配置某对象作为Array.prototype.concat()方法的参数时是否展开其数组元素。

代码示例

const alpha = ['a', 'b', 'c'];
const numeric = [1, 2, 3];
let alphaNumeric = alpha.concat(numeric);

console.log(alphaNumeric);
// Expected output: Array ["a", "b", "c", 1, 2, 3]

numeric[Symbol.isConcatSpreadable] = false;
alphaNumeric = alpha.concat(numeric);

console.log(alphaNumeric);
// Expected output: Array ["a", "b", "c", Array [1, 2, 3]]
