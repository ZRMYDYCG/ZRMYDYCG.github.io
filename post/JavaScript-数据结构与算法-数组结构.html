数组常用方法

push pop shift unshift splice

push 和 pop：从数组的后面追加和删除一个元素

unshift 和 shift：从数组的前面追加和删除一个元素

注意：pop() 和 shift() 的返回值是删除的数组元素

那咱要是想是想从任意位置进行元素的插入和删除操作呢？巧了，JavaScript正是提供了这样的一个数组的操作方法：splice()

语法

这个 splice( ) 方法的 ( ) 里接收 3 个参数

  1. 从第几个位置开始删除

  2. 要删除几个元素

  3. 这个部分的参数就可有意思了，可以进行追加，加入其他的数，加入一个数组形成数组的嵌套，其实追加对象也是可以的，谁叫你删了人家哟

代码示例

<script>
    let arr = new Array(1, 2, 3)
    arr.splice(0, 2, [5, 4], { name: '我也进来咯~' }, { age: '18岁了' })
    console.log(arr);
</script>


sort( ) 方法

sort( ) 方法中的 ( ) 接收一个回调函数

代码示例

<script>
    let arr = [2, 1, 4, 5, 7, 0]
    arr.sort((x, y) => x - y)

    let arr1 = [3, 1, 2, 5, 0, 8]
    arr1.sort((x, y) => y - x)
</script>

**注意:**里面的 x , y 是数组里面的每一个元素

那如果数组里面有几个学生信息的对象元素，咱想这些对象按年龄从小到大排序呢？

代码示例

<script>
    let arr = [{
        name: 'Zeng',
        age: 20
    }, {
        name: 'Ri',
        age: 18

    }, {
        name: ' Ming',
        age: 19
    }]

    // 以年龄的顺序进行排序

    arr.sort((x, y) => (x.age - y.age))
</script>

迭代方法 every , some , filter , map ,forEach , reduce

  1. every

这个 every 啊，返回的是布尔值，其实讲白辽就是判断数组中每一个元素是否都符合条件，如果都合她意了，就返回 true , 否者就是false。

这个 every 方法接收的是一个回调函数，回调函数中是咱传进去的参数，也就是数组中的每一项。

代码示例

<script>
    let arr = [10, 11, 12, 13, 14];

    console.log(arr.every(item => item > 1));  // true

    console.log(arr.every(item => item > 10));  // false
</script>
  1. some

这个 some 啊，和 every 就是一对嘛，但他要求就没拉么高了，只要数组中有一个元素符合他的条件，就返回 true , 否则就是 false。

代码示例

<script>
    let arr = [10, 11, 12, 13, 14];
    console.log(arr.some(item => item > 13));  // true
</script>
  1. filter

咱现在来看看这个 filter ，他呢？就是把符合条件的数组元素给筛选出来，并且返回一个新的数组，不光有并且，还有而且，他还不会改变原数组

代码示例

<script>
    let arr = [1, 2, 3, 4, 5, 6]

    let res = arr.filter(item => item > 5)

    console.log(res)  // 6
</script>
  1. map

map 是一个映射的作用,原数组真的是一点都改不了,但其映射出了一个新的数组

咱就是说既然这个 map 和 forEach 都是起一个遍历作用，连传递的参数都他娘的一样，那是不是map也可以拿来当forEach用，可以是可以哦，但是咱这不违背了设计者的初衷了嘛？哈哈！map可以理解为功能更完善的forEach

代码示例

<script>
    let arr = [1, 2, 3, 4, 5]
    let res = arr.map(item => item + "湖人总冠军！！！")
    console.log(res)
    console.log(arr)
</script>
  1. forEach

forEach 主打的就是一个遍历,也不会把原数组给改变了，他这里可以接收两个参数，第一个是遍历的每一个元素，第二个是元素下标，不过这个forEach和map的重点区别就在于，map有返回值，而forEach没有返回值

<script>
    let arr = [1, 2, 3, 4, 5]

    let doubleArr = []

    let indexArr = []

    arr.forEach((item, index) => {
        doubleArr.push(item * 2)
        indexArr.unshift(index)
    })

    console.log(doubleArr)

    console.log(indexArr)
</script>
  1. reduce

这个 reduce 迭代方法有撒子用哦？咱可以拿它来做一些累加累乘的操作，当然以后在实际开发中，它还会有很多应用

这 reduce 里面有两个参数 item1 , item2

代码示例

<script>
    let arr = [1, 2, 3, 4]

    let res = arr.reduce((item1, item2) => item1 + item2)

    console.log(res)
</script>

这个 item 1，item 2 一开始是接收了 arr 中的第一和第二个参数 item1 = 1, item2 = 2,将其执行函数体内 item1 + item2 操作, 其和作为 下一次操作的 item1 的值 , 再与 item2(item2 == 3 啦) 相加。诶哟~真想用我滴口才来表达这段文字,这文字太难表达辽！！！

搜索 indexOf , lastIndexOf , find , findIndex , findLast ,findLastIndex , includes

  1. indexOf 与 includes

这两个方法都是搜索数组中是否有这个值

indexOf 有：返回 0 没有：返回 -1

所以以后只要判断 返回值是否大于 -1 , 就可以知道该元素是否在数组里面了。

includes 有：返回 true 没有：返回 false

代码示例

<script>
    let arr = [1, 2, 3, 4]

    console.log(arr.indexOf(1))
    console.log(arr.indexOf(5))

    console.log(arr.includes(1))
    console.log(arr.includes(5))
</script>
  1. find findIndex findLast findLastIndex

  • find 查找并返回满足条件的第一个元素

  • findLast 倒序查找并返回满足条件的第一个元素，也即是最后一个满足条件的元素

  • findIndex 查找并返回满足条件的第一个元素的下标

  • findLastIndex 倒序查找并返回满足条件的第一个元素下标，也即是最后一个满足条件的元素下标

代码示例

<script>
    let arr = [1, 2, 3, 4, 5]
    let res1 = arr.find(item => item > 2)
    let res2 = arr.findIndex(item => item > 2)
    console.log(res1, res2)
    let res3 = arr.findLast(item => item > 2)
    let res4 = arr.findLastIndex(item => item > 2)
    console.log(res3, res4)
</script>

数组中的 slice 方法

slice() 方法返回一个新的数组对象，这一对象是一个由 start 和 end 决定的原数组的浅拷贝（包括 start，不包括 end），
其中 start 和 end 代表了数组元素的索引。原始数组不会被改变。

咱先来尝试一下下：

const animals = ['ant', 'bison', 'camel', 'duck', 'elephant'];

console.log(animals.slice(2));
// Expected output: Array ["camel", "duck", "elephant"]

console.log(animals.slice(2, 4));
// Expected output: Array ["camel", "duck"]

console.log(animals.slice(1, 5));
// Expected output: Array ["bison", "camel", "duck", "elephant"]

console.log(animals.slice(-2));
// Expected output: Array ["duck", "elephant"]

console.log(animals.slice(2, -1));
// Expected output: Array ["camel", "duck"]

console.log(animals.slice());
// Expected output: Array ["ant", "bison", "camel", "duck", "elephant"]
