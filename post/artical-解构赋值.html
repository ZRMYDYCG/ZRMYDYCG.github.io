解构赋值

解构赋值的意思就是快速地从对象或者数组中取出成员的一种语法方式

数组

原来我们来拿数组里每一项成员的值，可以通过下标来获取

新方法：解构

代码示例 👉

    let arr = [1, 2, 3]
    let [x, y, z] = arr
    console.log(x, y, z)  

---控制台---
1 2 3

数组解构赋值挺方便的，比如交换 x，y 的值

代码示例 👉

    let x = 1;
    let y = 2;
    [y, x] = [x, y]

回到最开始的例子里，如果咱只想拿到数组里边的3，该怎么写呢？

代码示例 👉

    let arr = [1, 2, 3]
    let [, , a] = arr
    console.log(a); 

---控制台---
3

那要是数组里面又嵌套了数组，该怎么处理呢？

代码示例 👉

    let arr = [1, [2, 3, 4], 5, 6]
    let [a, [b], c] = arr
    console.log(b) 

---控制台---
2

比如这个例子，咱就拿到了嵌套数组里的第一个元素

对象

对象解构与数组不同，它是无序的，是通过对应的 key 来寻找并 抽取 出来

代码示例 👉

    let obj = {
        name: '刘德华',
        age: '18'
    }
    // let { name, age } = obj
    let { age, name } = obj // 交换一下也是可以滴
    console.log(name, age);

---控制台---
刘德华 18

这样一来就在面对了一个对象中的大量数据，咱就可以通过解构的方式，通过 key 抓 到我们所需要的数据，是不是及其方便!

代码示例 👉

    let obj = {
        name: '日闽',
        tall: '180',
        weight: '75KG',
        face: '帅',
        money: '1000000000000万',
        address: {
            china: '北京10套 苏州园林28套',
            US: '欧洲农场20万亩',
        }
    }
    /* 将其解构取值 */
    let { face } = obj
    console.log(face);

    let { address } = obj
    console.log(address);

    let { china } = address
    console.log(china)

咱就说假如在外面早就定义了一个name，且是全局的，那这咋办？

解决方法

代码示例 👉

    let name = '刘泽泉'
    let obj = {
        name: '日闽',
        tall: '180',
        weight: '70KG',
        face: '帅',
        money: '1000000000000万',
        address: {
            china: '北京10套 苏州园林2套',
            US: '园林1万亩',
        }
    }
    let { name: newname } = obj ⭐⭐⭐
    console.log(newname);

---控制台---
日闽

嵌套

要是想获得list这个数组，咱先不用嵌套写法来获得

代码示例 👉

    let obj = {
        rel: '2',
        name: 'xiao',
        data: {
            list: ['a', 'b', 'c']
        }
    }
    let { data } = obj
    let { list } = data
    console.log(list)

用嵌套写法来获得

代码示例 👉

    let obj = {
        rel: '2',
        name: 'xiao',
        data: {
            list: ['a', 'b', 'c']
        }
    }
    let { data: { list } } = obj ⭐⭐⭐
    console.log(list)

咱要是想得到数组里面的 c 呢？那就继续嵌套呀！！！

代码示例 👉

    let obj = {
        rel: '2',
        name: 'xiao',
        data: {
            list: ['a', 'b', 'c']
        }
    }
    // 数组解构是下标一一对应，key 值不用相同
    let { data: { list: [, , z] } } = obj
    console.log(z)

---控制台---
c

函数参数解构赋值

当我们接到后端传来的数据后，我们希望通过函数的方式来处理数据，得到我们想要的部分。

    function getData() {
        let res = {
            code: 200,
            data: {
                list: ['aaa', 'bbb', 'ccc']
            }
        }
        test(res)
    }
    function test(res) {
        let { code } = res
        console.log(code);
    }
    getData()

---控制台---
200

在上部分函数中通过test传过来的对象可不可以在直接在接收函数的参数部分就进行处理呢？

当然是可以的！！！

代码示例 👉

    function getData() {
        let res = {
            code: 200,
            data: {
                list: ['aaa', 'bbb', 'ccc']
            }
        }
        test(res)
    }
    function test({ code, data }) {
        console.log(code, data);
    }
    getData()

---控制台---
200 object

把字符串当个数组就可以了

代码示例 👉

let name = 'xiaoriming'
let [x, y, z] = name
console.log(x, y, z);

---控制台---
